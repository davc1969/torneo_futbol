#Simulacion de torneo de futbol en JAVA

###Pendientes
- Generar algoritmo para crear los partidos dependeindo del número de equipos y de posibles partidos por jornada.  La idea es que en una jornada juegen todos los partidos que permitan la cantida de equipos.
- Generar rondas de partidos, es decir si el campeonato se juega a dos rondas (ida y vuelta9, el tota de partidos será el doble de si se juega una sola rondas
- Crear método para asignar probabilidades desde el teclado o desde archivo aparte, la idea es que se pueda actualzar sin cambiar el código
- Incorporar los goles máximo por equipo según las estadísticas.  No siempre un equipo va a meter 5 goles
